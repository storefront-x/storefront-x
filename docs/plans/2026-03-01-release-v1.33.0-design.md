# Release v1.33.0 — Vue 3.5 Upgrade

## What's Being Released

- Bump Vue from `^3.4.15` to `^3.5.0` in `modules/vue/package.json`
- Updated `yarn.lock`

## Version Semantics

- **1.33.0** (minor bump) — consistent with project history (v1.31.0 was "Upgrade dependencies")
- Vue 3.4 -> 3.5 is a minor Vue upgrade, backward-compatible for SFX users

## Release Process

### Prerequisites

1. Branch `vue-upgrade` merged into `main`
2. All tests pass (`yarn test:playwright`, `yarn test:unit`, `yarn lint`)
3. Build works (`yarn build --config storefront-x.magento.config.js`)
4. npm publish rights for `@magexo/*`

### Steps

**Phase 1: Preparation**
1. `git checkout main && git pull`
2. Verify clean state: `git status`

**Phase 2: Verification**
3. `yarn install`
4. `yarn lint`
5. `yarn build --config storefront-x.magento.config.js`
6. `yarn test:unit`
7. `yarn test:playwright`

**Phase 3: Version bump**
8. `npx lerna version 1.33.0 --conventional-commits --yes`
   - Bumps version in all 83 `modules/*/package.json` + `lerna.json`
   - Generates/updates `CHANGELOG.md`
   - Creates git commit + tag `v1.33.0`
   - Pushes to origin

**Phase 4: Publish**
9. `npx lerna publish from-git --yes`
   - Publishes all packages to npm

**Phase 5: GitHub Release**
10. Create GitHub Release for tag `v1.33.0`
    - Triggers `release.yml` workflow (CI verification)

## Tool: Lerna

- Lerna v6, fixed mode (`"independent": false`)
- All packages share the same version
- `lerna version` handles: package.json updates, CHANGELOG, commit, tag, push
- `lerna publish from-git` reads version from git tag and publishes to npm
